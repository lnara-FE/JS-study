## 💻 서버 실행하기!

#### 1. 저장소를 local에 clone 해줍니다.

```sh
git clone https://github.com/blackcoffee-study/moonbucks-menu-server.git
```

#### 2. 저장소를 clone한 후 아래 명령어를 입력해 의존성 라이브러리를 설치해줍니다.

```sh
npm install
```

#### 3. 아래 명령어를 이용해 서버를 동작시킵니다.

```sh
npm start
```
